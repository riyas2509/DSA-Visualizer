# 🧠 DSA Visualizer (Console-Based in C)

An interactive terminal-based Data Structures and Algorithms visualizer built in C.
This project demonstrates core DSA concepts using ASCII graphics and step-by-step sorting animations.

## 🚀 Features
### 🔹 Stack Visualizer
- Push, Pop, Display
- Overflow & Underflow handling
- Vertical ASCII representation

### 🔹 Queue Visualizer
- Enqueue, Dequeue, Display
- FIFO behavior visualization

### 🔹 Binary Search Tree Visualizer
- Insert nodes
- Inorder, Preorder, Postorder traversals
- Indented ASCII tree structure

### 🔹 Sorting Animation (Bubble Sort)
- Step-by-step comparison
- ASCII bar visualization
- Terminal-based animation with delay

## 🛠 Tech Stack
- Language: C
- Compiler: GCC
- Concepts Used: Arrays, Pointers, Structures, Dynamic Memory Allocation, Recursion, Modular Programming

## 📂 Project Structure
DSA-Visualizer/
├── main.c      → Menu & navigation
├── stack.c     → Stack implementation
├── queue.c     → Queue implementation
├── tree.c      → Binary Search Tree
├── sort.c      → Bubble sort animation
└── README.md

## ⚙️ Compile & Run

### Compile:
```bash
gcc main.c stack.c queue.c tree.c sort.c -o dsa_visualizer
```

### Run:

#### Windows:
```bash
.\dsa_visualizer
```

#### Linux/macOS:
```bash
./dsa_visualizer
```
